# Tasks

The shared repository intentionally does not include a `tasks/TODO.md` with
active work. Tasks are local and should not be overwritten during a merge.

Use the local durable-notes `tasks/TODO.md` for:

- active work;
- blockers;
- parked follow-ups;
- compatibility notes from root `CODEX_TODO.md`;
- links to Linear issues or PRs.

When merging this repo into a local durable-notes hierarchy, preserve the local
task file.
